# Palindrome

안녕하세요. 여러분 늦게 올려서 죄송합니다. 제가 이번에 낸 문제는 문자열이 펠린드롬인지 확인하는 간단한 예제입니다. 

주로 알고리즘 초보 영역에서 나오는 문제이니 어렵지 않을 것입니다.

## 펠린드롬 ? 

펠린드롬은 토마토, 실습실, eye, madam 같이 거꾸로 해도 단어가 같은 단어를 Palindrome 이라고 하고 한국어로 회문이라고 한답니다.
